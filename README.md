# dotfiles

## Keybinding grammar

- `hjkl` means direction.
- `Alt` is app-local options/alternate behavior, not a shared custom layer.
- Leader keys or key tables are for lifecycle commands like split, close, new, and zoom.
- Direct modifier chords move focus or siblings.
  - `Ctrl+hjkl` moves 2D panes/windows in the current app.
  - `Shift+h/l` moves 1D siblings like buffers, tabs, and pages.
- `Super` scopes the above ideas to the terminal emulator.
