TL;DR — cost-aware eval coverage

Add one reusable workflow scenario, a machine-readable fragment coverage catalog, and a no-execution campaign preview. Reuse the existing workload regression as the other workflow case. Run at most one treatment/control pair per workflow case initially, and repeat only if the paired evidence is ambiguous.
